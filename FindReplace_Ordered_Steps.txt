--- STEP 15: REGISTRY ROOT KEY CONSTANTS ------------------------------------

  Registry root key constants (HKEY_LOCAL_MACHINE, HKEY_CURRENT_USER, etc.)
  are declared As Long but must be passed to HKEY (LongPtr) parameters.

  Search your code for calls like:
    RegOpenKey HKEY_LOCAL_MACHINE, ...
    RegOpenKeyEx HKEY_CURRENT_USER, ...

  Wrap each root key constant with CLngPtr():
    RegOpenKey CLngPtr(HKEY_LOCAL_MACHINE), ...
    RegOpenKeyEx CLngPtr(HKEY_CURRENT_USER), ...

  OR add LongPtr variable declarations:
    Dim hRootKey As LongPtr
    hRootKey = CLngPtr(&H80000002)  ' HKEY_LOCAL_MACHINE
    hRootKey = CLngPtr(&H80000001)  ' HKEY_CURRENT_USER

AFTER ALL STEPS:
  1. VBA IDE → Debug → Compile (zero errors = success)
  2. If errors remain: the error line points to the problem
  3. Common post-fix errors:
     - "Type mismatch": Long variable receiving LongPtr function result
       → Find "Dim <varname> As Long" for that variable and change to LongPtr
     - "Expected: expression": Bad replace created syntax error
       → Undo (Ctrl+Z) that specific replace and redo manually
     - Comparison of LongPtr to 0: "If hWnd = 0" works fine — no change needed
     - Registry root key error: Wrap constant with CLngPtr() as shown in Step 15
